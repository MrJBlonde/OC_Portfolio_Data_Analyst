## 📂 Projets

### 🔷 P8 — Pipeline ETL & Analyse Sociodémographique ⭐ *Projet phare*
> **Stack : dbt Cloud · Snowflake · SQL · Python · INSEE**

**Contexte :** Data Analyst junior chez OpenClassrooms. Mission : analyser l'évolution du profil sociodémographique des étudiants Data sur 4 ans.

**Ce que j'ai fait :**
- Conception et déploiement d'un **pipeline de transformation dbt** documenté et reproductible
- Intégration dans **Snowflake** et déclaration des sources via fichiers `.yml`
- **Vérification qualité des données** et conformité **RGPD** (anonymisation, pseudonymisation)
- Enrichissement avec les **données publiques INSEE** pour contextualiser les résultats
- Analyse des indicateurs sociodémographiques : âge, genre, région sur 4 ans
- Restitution en 15 slides à destination d'un public non technique

**Livrables :** Pipeline dbt · Dataset `.csv` · Présentation stratégique
